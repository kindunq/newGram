# newGram
Instagram clone with Express + Prisma + React and React Native


## User Stories

- [] Like / unlike a photo
- [] Commnet on a photo
- [] Search by user
- [] Search by location
- [] See user profile
- [] Follow / Unfollow user
- [] See the full photo
- [] Edit my profile
- [] Create account
- [] Log in
- [] Upload photos
- [] Edit the photo(Delete)
- [] See the feed
